# Store UI

This is a Store UI, with an working chart, made with React, Redux and Tailwind CSS

## Author

- [@wdpedroborges](https://www.github.com/wdpedroborges)


## Features

- Add items to the chart
- Remove items from the chart
- Calculate the total price

## Tech

- React
- React Router
- Redux
- Tailwind

## Live demo

https://wdpedroborges.github.io/store-ui

## Installation

Install store-ui with npm

```bash
  git clone https://github.com/wdpedroborges/store-ui
  cd store-ui
  npm install
  npm run dev
```
    
## Build

```bash
  npm run build
```

## Deploy

```bash
  npm run deploy
```
